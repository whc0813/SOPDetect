项目名称：基于视觉分析的标准操作流程管理系统

项目类型：毕业设计

项目描述：本项目旨在开发一个基于视频分割和大模型识别的标准行为识别系统。
系统的主要功能包括：首先，接收标准行为规范输入，以定义待识别的行为标准；
其次，对输入的视频进行解析和处理，提取关键帧并进行视频分割，以便于后续分析；
然后，利用大模型对分割后的视频片段进行行为识别，准确识别出标准行为；
接着，系统将根据识别结果进行评分，评估行为的符合程度；
最后，提供反馈机制，向用户展示识别结果和评分，并提出改进建议，以优化行为表现。
通过这一系统，期望能够实现高效、准确的行为识别，推动相关领域的发展。

页面划分：登录页面、管理员端、用户端
功能：
管理员端：新建SOP->设定SOP步数->为每一步添加描述
用户端：选择SOP->上传每一步对应的视频->评估
